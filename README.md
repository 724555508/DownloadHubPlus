# DownloadHubPlus

一款轻量级的 Android 浏览器应用，自动检测网页中的视频资源（包括pornhub，xvideos，xxnn，eporner等）。

## 📱 主要功能

### 🌐 浏览器核心
- **快速浏览**：基于 WebView 的轻量级浏览器内核
- **多标签管理**：支持多标签页同时浏览
- **隐私模式**：无痕浏览，保护您的隐私
- **自定义搜索引擎**：支持 DuckDuckGo、Google、Bing 等搜索引擎

### 📥 下载管理
- **强大下载器**：内置专业下载管理器
- **视频嗅探**：自动检测网页中的视频资源
- **后台下载**：支持后台下载，不影响其他操作
- **文件管理**：便捷的下载文件管理功能

### 🎬 视频播放
- **内置播放器**：应用内直接播放视频
- **视频列表**：自动整理下载目录中的视频文件
- **小窗播放**：支持悬浮窗口播放

### 🔐 应用锁
- **密码保护**：支持应用锁功能
- **隐私安全**：保护您的浏览记录和应用数据
