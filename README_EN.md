# DownloadHubPlus

A lightweight Android browser application with automatic video detection from web pages (including pornhub, xvideos, xxnn, eporner, etc.).

## 📱 Key Features

### 🌐 Browser Core
- **Fast Browsing**: Lightweight WebView-based browser engine
- **Tab Management**: Support for multiple tabs
- **Privacy Mode**: Incognito browsing to protect your privacy
- **Custom Search Engines**: Supports DuckDuckGo, Google, Bing and more

### 📥 Download Manager
- **Powerful Downloader**: Built-in professional download manager
- **Video Detection**: Automatically detects videos on web pages
- **Background Downloads**: Download files without interrupting other activities
- **File Management**: Convenient download file management

### 🎬 Video Player
- **Built-in Player**: Play videos directly within the app
- **Video List**: Auto-organize videos from download directory
- **Popup Player**: Support floating window playback

### 🔐 App Lock
- **Password Protection**: App lock feature
- **Privacy Security**: Protect your browsing history and app data
